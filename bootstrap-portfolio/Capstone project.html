I-lms Project

Project Aim: This is a LMS(Learning management system project) that our college is currently using. But it was developed using Angular-Js We wanted to build it using React-Js. The main aim is to make the UI appealing so that the student finds it interesting.

New features: This has new features added such as:

  • Google signin
  • Student-level personalization
  • Different formats of quiz assesmants
  • Sem-wise IT and SS courses along with timeline for easy access

I was a part of program management team in which the landing pages of the mentor as well as the student will be displayed on logging in. Below is the sprint wise work that I have done.

Tools and Technologies used:
Tools:

  • Just In Mind for prototyping
  • Vs-code for coding
  • Google spread sheets for writing user stories
  • Postman for API calls
Technologies:
React-Js,JSX,CSS-3
Packages used:
Material UI, Bootstrap, font awesome, React Icons, cimpress,mui-treasury


Sprint1

Designing and prototyping

We as a team have identified the issues in the current LMS and wanted to make it more interactive and good looking. So we came up with a design and have prototyped the same using the tool Just In Mind
Impediments:
To understand and use the prototype tool was difficult in the start. But as we started using it was fun.

Sprint2

Program page

Here, I have developed a program page so that when ever a mentor logs in he/she gets to see all the programs that they are registered in. These programs are dynamically displayed in the form of cards. We used bootstrap for cards.
Impediments:
API call implementation.

Sprint3

Course page

This displays all the available courses in a particular program. Whenever a particular program is clicked it should be navigated to the respective course page. It differs for mentors and students.
Impediments:
Redirection was difficult onclick.

Sprint4

Cards and timeline

Here, I designed cards with the course image, start date and name. Various courses will be diaplayed across the page based on start and end dates. There is also a timeline section that indicates the current week number.

dynamic timeline implementation was difficult.

Sprint5

Card-fadeout

This displays the upcoming and ongoing courses in random colors. All the completed courses are faded out. A new UI is added to the dropdown button.
